# point-api

API RESTful para registro e gerenciamento de ponto eletrônico com fins de estudo. Funcionários registram entrada e saída; administradores acompanham relatórios e métricas de horas trabalhadas.

Tecnologias

Go 1.22
chi — roteador HTTP
sqlx — queries no PostgreSQL
JWT — autenticação stateless
bcrypt — hash de senhas
PostgreSQL — banco de dados


Pré-requisitos

Go 1.22+
PostgreSQL rodando localmente ou via Docker
